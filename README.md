# What says … ?

That's the question. Who says what ? Find the who and you'll see the what.

# Dependencies

Ruby of course.

# Usage

`./what_says <one you think to be the who>`

By example:

`./what_says Steve Jobs`

`./what says my little pony`

(this is not the correct answer…)

Enjoy!
